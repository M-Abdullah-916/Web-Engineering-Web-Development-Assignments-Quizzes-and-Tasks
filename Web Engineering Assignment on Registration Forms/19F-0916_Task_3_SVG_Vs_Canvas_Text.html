Difference Between SVG and Canvas


SVG Canvas
SVG stands for Scalable Vector Graphics
SVG is composed of shapes Canvas is composed of pixels
SVG is Vector based Canvas is Raster based
SVG has better Scalability Canvas has Poor Scalability
SVG can be printed with high quality at any resolution Canvas is not suitable to print on higher resolution
SVG provides better performance with smaller number of objects Canvas provides better performance with larger number of objects
SVG provides better performance with larger surface Canvas provides better performance with smaller surface
SVG can be modified with script or css Canvas can be modified with script only
SVG makes multiple graphical elements for a page Canvas makes single element with same behaviour
SVG has better text rendering capabilities Canvas has poor text rendering capabilities
